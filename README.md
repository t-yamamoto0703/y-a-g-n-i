# Y-A-G-N-I

## Overview

This project is a simple browser-based game themed around Japanese text input (IME).  
Players type the text displayed on the screen using their Japanese input method.

## Live Demo

- https://t-yamamoto0703.github.io/y-a-g-n-i/

## Development Goals

This project was created as part of a personal training workflow. Its objectives include:

- Practicing fundamental programming skills
- Training logical and structured thinking
- Making the development process visible and reviewable
- Exploring how generative AI can support software development

## Tested Environments

The game has been validated in the following environments:

- Lubuntu 24.04.4 LTS / Chromium
- Windows 11 / Microsoft Edge, Firefox

Debugging console output (F12) is intentionally left enabled.

## Assets and Licensing

- The background image was generated using AI.
- The source code is released under the MIT License.  
  Attribution is not required, as this type of game tends to converge to similar structural patterns regardless of the author.

## Additional Notes

- The project emphasizes “stopping at an appropriate point” rather than extensive polishing.  
  Generative AI was used throughout development, including code generation, state‑transition design, bug fixing, and refactoring.
- Future modifications or refactoring are undecided.
- For a basic Japanese typing game, the current level of complexity is considered sufficient as a starting point.
- The implementation is intentionally minimal and frozen in its present form as part of the learning process.
