# Y-A-G-N-I

## 概要
- 本作品は日本語入力（IME）をテーマにしたブラウザゲームです。

## 公開ページ
- [Y-A-G-N-I](https://t-yamamoto0703.github.io/y-a-g-n-i/)

## 開発目的
- 本プロジェクトは基礎的なプログラミングスキルおよび論理的思考の訓練、ならびに技術的理解の可視化を目的として作成されています。
- 本コードは「作り込む」ことよりも適切な段階で「切り上げる」ことを重視し、新規コード生成、状態遷移図の作成、バグ修正、リファクタリングなど、多くの開発プロセスにおいて生成AIを活用しながら進めました。

## 動作環境
以下の環境でのみ簡易的な動作確認を行っています。

- Lubuntu 24.04.4 LTS / Chromium
- Windows 11 / Microsoft Edge, Firefox

デバッグ用のコンソール出力を残していますので、必要に応じてご利用ください。

## 補足
- 背景画像は生成AIによって作成しています。
